# 🧮 Simple Command Line BMI Calculator

A lightweight Python-based Command Line Interface (CLI) application to calculate your **Body Mass Index (BMI)** quickly and easily.

---

## 📌 Features

* Simple and user-friendly CLI
* Takes height and weight as input
* Calculates BMI instantly
* Displays BMI category:

  * Underweight
  * Normal weight
  * Overweight
  * Obese

---

## ⚙️ How BMI is Calculated

BMI is calculated using the formula:

```
BMI = weight (kg) / (height (m))²
```

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Mousam-gift/simple-BMI-calculator.git
cd simple-BMI-calculator
```

---

### 2. Run the Program

Make sure you have Python installed.

```bash
python calculator.py
```

---

## 🧑‍💻 Usage

1. Enter your weight in kilograms
2. Enter your height in meters
3. Get your BMI and category instantly

### Example:

```bash
Enter your weight (kg): 70
Enter your height (m): 1.75

Your BMI is: 22.86
Category: Normal weight
```

---

## 📊 BMI Categories

| BMI Range   | Category      |
| ----------- | ------------- |
| < 18.5      | Underweight   |
| 18.5 – 24.9 | Normal weight |
| 25 – 29.9   | Overweight    |
| ≥ 30        | Obese         |
